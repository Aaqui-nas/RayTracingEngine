# TP19 — Éclairage direct (NEE)
#
# Cette scène teste le shadow ray et le Next Event Estimation :
#   - Deux lumières d'aire sphériques de couleurs différentes
#   - Sphères diffuses pour observer les ombres douces et la contribution de chaque lumière
#   - Sol réfléchissant pour observer les reflets de lumière
#
# Compare le bruit avec et sans NEE :
#   ./build/raytracer 1.0 32 scenes/scene_direct.txt > outputs/nee_32.ppm
#
# Avec NEE, 32 samples ≈ 256 samples sans NEE pour les surfaces diffuses.
