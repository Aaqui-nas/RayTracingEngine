# TP23 — Paysage procédural

# Terrain 32×32 généré avec bruit de Perlin.
# Format terrain : N M cell_size height_scale noise_scale noise_amplitude mat
#   N, M            : taille de la grille (nombre de cellules)
#   cell_size       : taille d'une cellule en unités monde
#   height_scale    : multiplicateur vertical global
#   noise_scale     : fréquence du bruit (plus grand = détail plus fin)
#   noise_amplitude : amplitude crête-à-crête du bruit de Perlin
#
# Le terrain part de (0, 0, 0) et s'étend vers +x et +z.
# Avec 32 cellules à cell_size=0.6 : terrain 19.2 × 19.2 unités.
#
# Rendu :
#   ./build/raytracer 1.0 64 scenes/scene_landscape.txt > outputs/landscape.ppm
