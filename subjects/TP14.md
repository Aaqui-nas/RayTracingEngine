# TP14 — Accélération : AABB et BVH

## Objectif

Sans optimisation, ton ray tracer teste **chaque objet pour chaque rayon**. Avec 50 000 triangles,
ça fait 50 000 tests × 800 000 pixels × 64 samples = **2,5 milliards d'intersections** par image.
À 10 ns par test, c'est **7 heures** de rendu.

Avec un **BVH (Bounding Volume Hierarchy)**, on descend à ~17 tests par rayon (log₂ 50 000). Résultat : **quelques minutes**.

**Point de départ** : le code issu de TP13 (avec `triangle.h`, `mesh.h`, `obj_loader.h`).

---

## 1. AABB — Boîte englobante alignée sur les axes

Une **AABB** (Axis-Aligned Bounding Box) est une boîte délimitée par deux points : `min` et `max`.

```
     max
      +──────+
     /      /|
    +──────+ |
    |      | +
    |      |/
    +──────+
   min
```

Si un rayon ne touche pas la boîte englobante d'un objet, il n'est **pas possible** qu'il touche l'objet à l'intérieur. On court-circuite le test coûteux.

### Méthode des dalles (slab method)

L'idée : tester l'intersection avec une boîte revient à tester 3 paires de plans (x_min/x_max, y_min/y_max, z_min/z_max) et trouver l'intervalle [t_entrée, t_sortie].

Pour chaque axe `i` (0=x, 1=y, 2=z) :
```
t0 = (min[i] - ray.origin[i]) / ray.direction[i]
t1 = (max[i] - ray.origin[i]) / ray.direction[i]
si t0 > t1 : échanger t0 et t1
tmin = max(tmin, t0)
tmax = min(tmax, t1)
si tmax <= tmin : pas d'intersection
```

Cette notation `ray.origin[0]` (accès par indice) sera rendue possible en ajoutant `operator[]` à `Vec3`.

### Interface de l'AABB

Crée `aabb.h` :

```cpp
#pragma once
#include "ray.h"

namespace rt {

struct AABB {
    Vec3d min_pt, max_pt;

    AABB() {}
    AABB(Vec3d mn, Vec3d mx) : min_pt(mn), max_pt(mx) {}

    bool hit(const Ray& ray, double tmin, double tmax) const;
};

inline AABB surrounding_box(AABB a, AABB b);

} // namespace rt
```

## 2. BVH — Arbre de boîtes englobantes

Le BVH est un **arbre binaire** où :
- Chaque feuille = un objet (Sphere, Triangle, Mesh...)
- Chaque nœud interne = une AABB qui contient ses deux enfants
- La racine = l'AABB de toute la scène

```
         [scène entière]
        /               \
   [gauche]         [droite]
   /      \          /     \
 sph1    sph2     tri1    mesh
```

**Traversal** : pour chaque rayon
1. Tester l'AABB du nœud actuel
2. Si miss → retourner nullopt (économise tous les tests des enfants)
3. Si hit → tester les deux enfants récursivement, retourner le plus proche

**Construction** : algorithme récursif
1. 1 objet → `left = right = cet objet`
2. 2 objets → `left = obj[0]`, `right = obj[1]`
3. n objets → trier par centroïde sur un axe (choisi aléatoirement), couper au milieu, récursion

### Interface du BVHNode

Crée `bvh.h` :

```cpp
#pragma once
#include "shape.h"
#include "aabb.h"
#include <vector>
#include <memory>
#include <algorithm>
#include <random>

namespace rt {

class BVHNode : public Shape {
public:
    std::shared_ptr<Shape> left;
    std::shared_ptr<Shape> right;
    AABB box;

    BVHNode(std::vector<std::shared_ptr<Shape>>& shapes, int start, int end);

    std::optional<HitRecord> hit(const Ray& ray, double tmin, double tmax) const override;
    std::optional<AABB> bounding_box() const override { return box; }
};

} // namespace rt
```

## 3. Boîte englobante par type de Shape

Chaque `Shape` doit savoir calculer sa propre boîte. On ajoute une méthode virtuelle à la classe de base :

| Shape    | AABB                                            |
|----------|-------------------------------------------------|
| Sphere   | `[center - radius, center + radius]`            |
| Triangle | `[min(A,B,C), max(A,B,C)]` + epsilon            |
| Mesh     | union de toutes les boîtes de ses triangles     |
| Plane    | infinie → `nullopt` (pas dans le BVH)           |

---

## 4. Intégration dans la scène

`Scene::build_bvh()` sépare les objets en deux groupes :
- **Bornés** (Sphere, Triangle, Mesh) → enveloppés dans un BVHNode
- **Infinis** (Plane) → gardés dans `objects` séparément

Après construction, le `hit()` de la scène teste d'abord le BVH, puis les infinis.

std::optional<AABB> bounding_box() const override {
    Vec3d mn(std::min({A.x, B.x, C.x}), std::min({A.y, B.y, C.y}), std::min({A.z, B.z, C.z}));

}

std::optional<AABB> bounding_box() const override {
    if (triangles.empty()) return std::nullopt;

### Étape 7 — Créer `bvh.h` avec `BVHNode`

Crée `bvh.h` avec la classe de la Section 2.

**Implémenter `BVHNode::hit()`** :
1. Si `!box.hit(ray, tmin, tmax)` → retourner `std::nullopt`
2. Appeler `left->hit(ray, tmin, tmax)` → si trouvé, `tmax = hit_left.t`
3. Appeler `right->hit(ray, tmin, tmax)`
4. Retourner le résultat le plus proche

**Implémenter le constructeur** :

```
BVHNode(shapes, start, end) :
  int n = end - start
  if n == 1 :
    left = right = shapes[start]
  elif n == 2 :
    left = shapes[start], right = shapes[start+1]
  else :
    // Trier shapes[start..end] par centroïde sur un axe aléatoire (0, 1 ou 2)
    // L'axe peut être choisi au hasard : int axis = rand() % 3;
    std::sort(shapes.begin()+start, shapes.begin()+end, comparateur_par_axe)
    int mid = start + n/2
    left  = make_shared<BVHNode>(shapes, start, mid)
    right = make_shared<BVHNode>(shapes, mid, end)
  
  // Calculer la boîte englobante :
  auto lb = left->bounding_box()
  auto rb = right->bounding_box()
  box = surrounding_box(lb.value(), rb.value())
```

Le comparateur tri les shapes par la coordonnée minimum de leur boîte sur un axe donné :
```cpp
auto cmp = [axis](const auto& a, const auto& b) {
    return a->bounding_box().value().min_pt[axis]
         < b->bounding_box().value().min_pt[axis];
};
```

### Étape 8 — Ajouter `build_bvh()` à Scene

Dans `scene.h`, ajoute la méthode `build_bvh()` à la classe `Scene` :

```cpp
#include "bvh.h"

// Dans Scene :
std::shared_ptr<BVHNode> bvh_root;

void build_bvh() {
    std::vector<std::shared_ptr<Shape>> bounded;
    std::vector<std::shared_ptr<Shape>> unbounded;
    
    for (auto& obj : objects) {
        if (obj->bounding_box().has_value())
            bounded.push_back(obj);
        else
            unbounded.push_back(obj);
    }
    
    objects = unbounded;  // garde seulement les infinis dans objects
    if (!bounded.empty())
        bvh_root = std::make_shared<BVHNode>(bounded, 0, bounded.size());
}
```

Et modifie `Scene::hit()` pour tester aussi le BVH :

```cpp
std::optional<std::pair<HitRecord, int>> hit(const Ray& ray, double tmin, double tmax) const {
    std::optional<HitRecord> best;
    int best_idx = -1;
    double closest = tmax;
    
    // Tester les objets infinis
    for (int i = 0; i < (int)objects.size(); i++) {
        if (auto h = objects[i]->hit(ray, tmin, closest)) {
            closest = h->t;
            best = h;
            best_idx = i;
        }
    }
    
    // Tester le BVH
    if (bvh_root) {
        if (auto h = bvh_root->hit(ray, tmin, closest)) {
            best = h;
            best_idx = (int)objects.size();  // indice fictif pour le BVH
        }
    }
    
    if (!best) return std::nullopt;
    return std::make_pair(*best, best_idx);
}
```

> Note : l'indice retourné pour les objets dans le BVH est fictif ici, mais l'émission est stockée dans le HitRecord — assure-toi que le material est bien copié dans le HitRecord lors du hit.
