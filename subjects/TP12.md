# TP12 — Lambdas et C++ Moderne (C++17/20)

## Objectif final
Moderniser le code du ray tracer avec les fonctionnalités C++ récentes, et ajouter un système de matériaux avec des lambdas comme fonctions de couleur.

---

## 1. Lambdas

Une lambda est une fonction anonyme définie en ligne :

```cpp
auto add = [](int a, int b) { return a + b; };
std::cout << add(3, 4);  // 7
```

Capture de variables du contexte :

```cpp
int offset = 10;
auto add_offset = [offset](int x) { return x + offset; };  // capture par valeur
auto add_ref    = [&offset](int x) { return x + offset; };  // capture par référence
```

Avec `std::function` pour stocker des lambdas :

```cpp
#include <functional>
std::function<Vec3(const Ray&)> color_fn = [](const Ray& r) {
    return Vec3(1, 0, 0);  // rouge
};
```


---

## 4. `if constexpr` (C++17)

Branche de compilation conditionnelle :

```cpp
template<typename T>
void print(T val) {
    if constexpr (std::is_integral_v<T>) {
        std::cout << "int: " << val;
    } else {
        std::cout << "float: " << val;
    }
}
```

---

## 5. `std::optional` (C++17)

Pour retourner "quelque chose ou rien" sans valeur sentinelle :

```cpp
#include <optional>

std::optional<double> hit_sphere(const Ray& ray) {
    // ...
    if (no_hit) return std::nullopt;
    return t;
}

if (auto t = hit_sphere(ray)) {
    // utilise *t
}
```

## Exercice

1. Remplace le retour `bool hit(..., double& t)` par `std::optional<double> hit(...)` dans `Shape`, `Sphere`, `Plane`, `Scene`
2. Ajoute un système de matériaux : chaque `Shape` a un `std::function<Vec3(const Ray&, double t)> material` qui calcule la couleur
3. Utilise des lambdas pour définir les matériaux dans `main.cpp` :
   - Sphère rouge : `[](const Ray&, double) { return Vec3(255, 0, 0); }`

4. Utilise `auto` partout où le type est évident
5. Utilise un structured binding quelque part

